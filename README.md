
# Portfolio Website

A portfolio website built using HTML and CSS.  
It includes two pages: Home and Contact.

This project is created to demonstrate basic front-end development skills with a clean and simple design.

## Pages
- Home – Introduction and basic information
- Contact – Contact details

## Technologies Used
- HTML
- CSS

## Features
- Simple and clean layout
- Two-page structure
- Beginner-friendly design

## Purpose
The purpose of this project is to practice:
- Structuring web pages with HTML
- Styling with CSS
- Creating and linking multiple pages
